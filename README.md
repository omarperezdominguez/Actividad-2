Este repositorio contiene el modelado matemático y la validación cinemática de dos configuraciones de manipuladores robóticos de 6 Grados de Libertad (6GDL):

1. Robot Antropomórfico

2. Robot Industrial

El objetivo principal es determinar la posición y orientación de cada eslabón mediante el uso de Matrices de Transformación Homogénea, permitiendo conocer la ubicación del efector final respecto a la base del robot.
